# Colors

All colors are **OKLCH** — perceptually uniform, better for dark-mode contrast ratios than HSL.

Edit once in `src/styles/tokens.css`; every component and every Tailwind utility updates in lockstep.

## Tailwind mapping

Core colors (base, text, accent, feedback) are aliased into Tailwind utilities
via `styles/theme.css`. Pipeline-state colors stay as raw `var(--state-…)`
consumed from primitives or the `Badge` component — the `color-mix` tints live
in CSS, not in per-JSX-callsite utility classes. Examples of what's exposed:

```tsx
<div className="bg-surface border border-border text-text" />
<button className="bg-accent text-accent-ink" />
<span className="text-danger" />
```

State palette stays on `Badge tone="state-in-dev"` / `var(--state-in-dev)` in CSS.
