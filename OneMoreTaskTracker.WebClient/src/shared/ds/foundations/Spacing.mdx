## Motion

The system uses exactly two durations — no custom one-offs.

- `--transition-fast` = **120ms** — hover/active, feedback
- `--transition-base` = **160ms** — layout changes, state changes

> When something moves, it means something changed.
> Motion carries meaning; decoration does not.
